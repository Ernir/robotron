CONTROLS

Movement

  • W = move up
  • A = move left
  • S = move down
  • D = move right

Fire weapon

  • Arrow keys = fire in the arrow's direction
  • mouse movement = aim
  • mouse button = fire

Toggles

  • M = Music on/off (not yet implemented)
  • N = Sound effects on/off (not yet implemented)
  • R = Restart game
  • P = Pause the game
  • O = When paused - continue 1 frame
  • Q = Quit

Diagnostics

  • + = Next level
  • - = Previous level>/li>
  • K = Invincible mode
  • C = Clear the screen on/off
  • B = Draw a red box
  • U = Draw a white box
  • F = Toggle flipflop
  • R = Toggle render (toggleable only when paused)
  • T = Show timer
